Método de biscección.

i: Contador principal.
ite: Número de iteraciones máximas.
a, b, c: Variables tipo doble.
fa: Función evaluada en a.
fb: Función evaluada en b.
fc: Función evaluada en c.
err: Error absoluto calculado.
errLim: Error absoluto mínimo.
ValorAnterior: Valor de c en la última iteración.
Suponga la función f(x) = sen(x) + 3cos(x)

Código en Java:

for (i = 0; i < ite; i++) {
fa = Math.sin(a) + 3 * Math.cos(a);
fb = Math.sin(b) + 3 * Math.cos(b);
c = (a + b) / 2;
fc = Math.sin(c) + 3 * Math.cos(c);
if (Math.signum(fc) == Math.signum(fa)) {
a = c;
} else {
b = c;
}
if (i != 0) {
err = Math.abs(c - ValorAnterior);
if(err <= errLim) {
break;
}
} else {
tblDatos.setValueAt(null, i, 6);
}
ValorAnterior = c;
}

Método de Gauss.

A: Matríz de nxn.
i, j, k, l: Contadores.
n: Tamaño de la matriz.
aux: Variable doble auxiliar.
signo: Variable que guarda el signo del elemento a eliminar.
signoAux: Variable que guarda el signo del elemento pivote.
Código en Java:

for (int j = 0; j < n; j++) {
if (A[j][j] == 0D) {
if (j < n - 1) {
for (int k = j + 1; k < n; k++) {
if (A[k][j] != 0) {
for (int l = 0; l < n + 1; l++) {
aux = A[j][l];
A[j][l] = A[k][l];
A[k][l] = aux;
}
break;
}
}
}
}
for (int i = j + 1; i < n; i++) {
signo = Math.signum(A[i][j]);
signoAux = Math.signum(A[j][j]);
aux = Math.abs(A[i][j]);
if (signo == 1.0D && signoAux == 1.0D) {
for (int k = j; k < n + 1; k++) {
A[i][k] = A[i][k] - (aux / Math.abs(A[j][j])) * A[j][k];
}
} else if (signo == 1.0D && signoAux == -1.0D) {
for (int k = j; k < n + 1; k++) {
A[i][k] = A[i][k] + (aux / Math.abs(A[j][j])) * A[j][k];
}
} else if (signo == -1.0D && signoAux == 1.0D) {
for (int k = j; k < n + 1; k++) {
A[i][k] = A[i][k] + (aux / Math.abs(A[j][j])) * A[j][k];
}
} else if (signo == -1.0D && signoAux == -1.0D) {
for (int k = j; k < n + 1; k++) {
A[i][k] = A[i][k] - (aux / Math.abs(A[j][j])) * A[j][k];
}
}
}
}

i: Contador principal.
ite: Número de iteraciones máximas.
a, b, c: Variables tipo doble.
fa: Función evaluada en a.
fb: Función evaluada en b.
fc: Función evaluada en c.
err: Error absoluto calculado.
errLim: Error absoluto mínimo.
ValorAnterior: Valor de c en la última iteración.
Suponga la función f(x) = x^2 - 5x + 2
